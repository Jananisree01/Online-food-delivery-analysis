&nbsp;Online Food Delivery Analysis



This project is an end-to-end data analytics solution built to analyze online food delivery operations and generate actionable business insights.



The objective of the project is to understand customer behavior, revenue trends, delivery performance, restaurant performance, and profitability impact using real-world data analysis techniques.



The project follows a complete data pipeline:



Raw Dataset → Data Cleaning (Python) → MySQL Database → SQL Analysis → Interactive Dashboard (Streamlit)



***Tools Used***



\- Python

\- Pandas

\- MySQL

\- SQL

\- Streamlit



&nbsp;***Project Steps***



1\. Data Cleaning using Python

2\. Data stored in MySQL database

3\. SQL queries for analysis

4\. Interactive Dashboard using Streamlit

Key Analysis Performed

 Customer Analysis
- Top spending customers
- Age group vs average order value

 Revenue & Profit Analysis
- Monthly revenue trends
- City-wise revenue
- Cuisine-wise performance
- Impact of discounts on profit margin

 Delivery Performance
- Delivery time by city
- Delivery rating vs delivery time
- Distance vs delivery time

 Restaurant Performance
- Top rated restaurants
- Cuisine-wise rating analysis

 Operational Insights
- Peak hour demand
- Payment mode preference
- Cancellation analysis

***How to Run the Dashboard***



Install required libraries:



pip install -r requirements.txt



Run the app:



streamlit run app.py
